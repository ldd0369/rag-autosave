■ 작동 흐름
LibreChat 새 대화 → MongoDB conversations/messages 저장 (LibreChat 자동)
        ↓
1시간마다 rag-autosave 자동 실행
        ↓
새 메시지 감지 (rag_autosave_state.last_run 이후)
        ↓
대화 텍스트 조립 (conv_xxxx.txt) → RAG API /embed
   (multipart: file + file_id + user + entity_id=AGENT_ID) ★
        ↓
files 컬렉션에 레코드 INSERT (register_file 함수)
        ↓
rag_file_ids 컬렉션 기록
        ↓
루프 종료 후 agents.tool_resources.file_search.file_ids 일괄 업데이트 (1회) ★
        ↓
다음 질문부터 에이전트가 File Search tool로 entity_id 필터링해서 검색
